
# 🖐 Real-Time Hand Gesture Recognition System

> **B.Tech Final-Year AI/ML Portfolio Project**  
> Python · OpenCV · MediaPipe · TensorFlow/Keras · Streamlit

---

## Overview

A complete end-to-end deep-learning pipeline that detects and classifies hand gestures in real time using a standard webcam.  The system extracts 21 3-D hand landmarks via **MediaPipe**, normalises them into a compact 63-feature vector, and feeds the vector to a lightweight **dense neural network** trained with TensorFlow/Keras.  Results are visualised both in a standalone OpenCV window and in an interactive **Streamlit** web application.

---

## Features

| Feature | Detail |
|---|---|
| **6 gesture classes** | Open Palm, Fist, Thumbs Up, Thumbs Down, Victory, OK |
| **Real-time inference** | 25–60 FPS on a standard laptop CPU |
| **Multiple hands** | Up to 2 hands tracked simultaneously |
| **Confidence threshold** | Adjustable — uncertain predictions flagged visually |
| **Streamlit UI** | Browser-based dashboard with live feed & stats |
| **FPS counter** | Rolling 30-frame average displayed on every frame |
| **Training diagnostics** | Accuracy / loss curves + confusion-matrix PNG |
| **Modular codebase** | PEP 8, docstrings, type hints throughout |

## Architecture

```
Webcam frame
    │
    ▼
MediaPipe Hands  →  21 landmarks × (x, y, z)  =  63 raw features
    │
    ▼
Normalisation  (translate to wrist origin, scale to [-1, 1])
    │
    ▼
Dense NN Classifier
  Input(63)
  Dense(256, ReLU) → BatchNorm → Dropout(0.4)
  Dense(128, ReLU) → BatchNorm → Dropout(0.3)
  Dense( 64, ReLU) → BatchNorm → Dropout(0.2)
  Dense(n_classes, Softmax)
    │
    ▼
Predicted Gesture + Confidence
```

## Installation

```bash

cd hand-gesture-recognition
```

### 2. Create a virtual environment

## Usage

### Step 1 — Collect data

Run for **each** gesture class.  The script opens your webcam; press **SPACE** to save a sample.

```bash
python collect_data.py --gesture open_palm   --samples 300
python collect_data.py --gesture fist        --samples 300
python collect_data.py --gesture thumbs_up   --samples 300
python collect_data.py --gesture thumbs_down --samples 300
python collect_data.py --gesture victory     --samples 300
python collect_data.py --gesture okay        --samples 300
```

All samples are appended to `dataset/landmarks.csv`.

### Step 2 — Preprocess

```bash
python preprocess.py
```

Outputs: `dataset/X_train.npy`, `X_test.npy`, `y_train.npy`, `y_test.npy`, `models/label_encoder.pkl`.

### Step 3 — Train

```bash
python train_model.py --epochs 100 --batch 32
```

Outputs: `models/gesture_model.h5`, `models/confusion_matrix.png`, `models/training_curves.png`.

### Step 4a — Standalone OpenCV window

```bash
python predict.py --camera 0 --threshold 0.6 --max-hands 2
```

| Key | Action |
|-----|--------|
| `q` | Quit |
| `+` | Raise confidence threshold |
| `-` | Lower confidence threshold |

### Step 4b — Streamlit web app

```bash
streamlit run app.py
```

Open [http://localhost:8501](http://localhost:8501) in your browser, toggle **Start / Stop Camera**, and start gesturing.
